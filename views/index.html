Fake App DataBase

Una base de datos de aplicaciones falsas para todo tipo de tareas y proyectos escolares, libre de derechos para ser usadas en cualquier lugar y ocasión.

La base de datos esta conformada por 5 campos: el nombre de la Aplicación, el nombre del Desarrollador, la fecha en que se liberó, la compañía creadora o patrocinadora y la ultima versión liberada.
